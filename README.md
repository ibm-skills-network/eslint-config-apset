# eslint-config-apset
Node Style Guide for IBM Dev Skills Network.

## Install

```bash
npm i eslint eslint-config-apset
```

OR

```bash
yarn add eslint eslint-config-apset
```

## Usage
* Create `.eslintc` file
```json
{
  "extends": "eslint-config-apset"
}
```
